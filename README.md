This repository documents my journey of mastering Object-Oriented Programming (OOP) in Java through structured daily practice and TCS IPA-style coding challenges.

* ✅ Classes & Objects
* ✅ Constructors
* ✅ Methods
* ✅ Static Variables & Static Methods
* 🔄 Arrays of Objects
* ⏳ Encapsulation
* ⏳ Inheritance
* ⏳ Method Overloading
* ⏳ Method Overriding
* ⏳ Runtime Polymorphism
* ⏳ Abstraction
* ⏳ Interfaces
* ⏳ Final TCS IPA OOP Assessment

## Repository Structure

Each day's practice is organized into a separate folder containing:

* Problem statement implementation
* Java solution
* Python solution (where applicable)

## Goal

Build a strong OOP foundation and solve TCS IPA interview-style coding problems with confidence through consistent daily practice.
